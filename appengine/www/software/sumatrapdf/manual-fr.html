Paramètres de la ligne de commandes

Pour une documentation plus complète, lire cette page (en anglais).

Paramètres :

-page $numpage  Ouvre le document à la page spécifiée
-named-dest $dest  ouvre le document à la page de la destination (dans un document PDF, équivalent d'un signet)
-lang $language  Sélection de la langue pour l'interface utilisateur.
$language est un code de deux lettres, p. ex. "en", "de" (Voir la liste des codes
-restrict  Démarrer en mode restreint.
-reuse-instance  Utilise le programme en cours d'exécution au lieu de lancer une nouvelle instance
-presentation $file.pdf  Ouvre un fichier PDF en mode présentation (plein écran)
-register-for-pdf  Le programme définit Sumatra comme lecteur PDF par défaut puis se termine
-esc-to-exit  La touche ESC est définie comme touche de sortie du programme.
À n'utiliser qu'une fois, après cela le changement est enregistré dans le fichier de paramètres
-bg-color $couleur  change la couleur d'arrière-plan jaune en celle passée en paramètre.
$couleur est une valeur au format hexadécimal (p. ex. 0xffff00)
-invert-colors  Inverse les couleurs dans le document PDF (p. ex. le blanc devient noir, et le noir devient blanc).

Note: les options -esc-to-exit et -bg-color changent les valeurs par défaut et sont enregistrées dans le fichier de paramètres sumatrapdfprefs.dat (dans le répertoire %appdata%\SumatraPDF si installé avec le programme d'installation, sinon, dans le même répertoire que celui du programme).
Pour revenir aux paramètres par défaut, il suffit de supprimer le fichier sumatrapdfprefs.dat.

Paramètres pour l'impression :

-print-to-default $file.pdf  Imprime le fichier spécifié sur l'imprimante par défaut
-print-to $printer_name $file.pdf  Imprime le fichier spécifié sur l'imprimante désignée
-print-dialog $file.pdf  Ouvre le fichier spécifié et lance la boîte de dialogue pour le choix de l'imprimante et des options d'impression
-exit-on-print  quand -print-to-default ou -print-to est utilisé, quitte le programme à la fin de l'impression

Plugin (module) pour navigateurs Internet

Le programme d'installation de Sumatra inclut un plugin pour Firefox, Chrome et Opera (Internet Explorer n'est pas pris en charge). Il n'est pas installé par défaut, et il vous faut donc cocher l'option durant l'installation pour l'activer.

Vous pouvez avoir besoin de configurer le navigateur aussi, activer le plugin Sumatra et désactiver ceux des autres programmes.

Firefox

Vous devez désactiver les plugins PDF pour d'autres programmes (comme Adobe Reader ou Foxit) s'ils sont installés. Passez par le menu Outils/Modules complémentaires. Basculez sur l'onglet Plugins, activez le plugin Sumatra PDF et désactivez les autres plugins PDF.

Problèmes connus avec le plugin Firefox

Dans certains cas (quand le plugin Foxit est aussi déjà installé) le plugin Sumatra ne marchera pas même si les autres sont désactivés. Ce problème a été résolu avec Firefox 4.0.

Chrome

Les versions les plus récentes de Chrome ont un plugin PDF préinstallé. Si vous désirez utiliser celui de Sumatra, vous devez désactiver celui de Chrome (et possiblement ceux d'autres programmes, comme Adobe Reader ou FoxIt Reader).

Entrez about:plugins dans la barre d'adresse, puis assurez-vous d'activer le plugin PDF de Sumatra et de désactiver les autres.

Opera

  1. Démarrez Opera et tapez CTRL + F12
  2. Sélectionnez l'onglet Avancé (Advanced)
  3. Dans la liste sur la gauche, choisissez Téléchargements (Downloads)
  4. Dans la boîte de dialogue qui s'affiche, recherchez "application/pdf",
    cliquez desssus avec le bouton droit de la souris puis sélectionnez "Editer" (Edit)
  5. Sélectionnez "Utiliser le plugin" (Use the plugin) en bas et choisissez "SumatraPDF Browser Plugin"
  6. Cliquez sur "OK" puis de nouveau sur "OK". Redémarrez Opera

Démarrer en mode restreint

Le mode restreint est très utile si vous utilisez SumatraPDF comme visualiseur par défaut pour documenter ses programmes ou en mode «kiosque» (mode présentation). En mode restreint, certaines actions non adaptées à ces contextes sont désactivées :

  • Ouvrir de nouveaux fichiers
  • Ouvrir un page Web depuis un document PDF
  • Sélection d'images et de texte
  • Impression
  • Modifier les paramètres par défaut
  • Enregistrer le document
  • Vérification automatique et manuelle des mises à jour
  • Historique des derniers fichiers ouverts
  • support de TeX
  • Définir Sumatra comme lecteur PDF par défaut
  • Ouvrir le document avec Adobe Acrobat
  • Envoyer le document PDF par email

Notes

Si "Se rappeler ces paramètres pour chaque document" est coché dans les options, quand un document PDF est réouvert, Sumatra PDF ira directement à la dernière page affichée lors de sa dernière ouverture. Cela s'applique aussi aux documents déjà ouverts qui ne sont pas dans la liste des dix plus récents. Cette fonction ne s'appliquera pas à un document déjà ouvert mais qui aura été déplacé entretemps.

Certains paramètres (p. ex. le mode plein écran) sont enregistrés comme paramètres par défaut même quand ils n'ont pas été définis comme tels quand on quitte SumatraPDF sans qu'un document soit ouvert.

Restrictions connues

L'édition de formulaires interactifs et l'ajout de commentaires ne sont pas implémentés.
