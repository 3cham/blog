Sumatra PDF est un lecteur PDF, ePub, MOBI, CHM, XPS, DjVu, CBZ, CBR pour Windows

Historique

2.1.1 (2012-05-07)

Changements dans cette version:

  • Correction de quelques bugs
2.1 (2012-05-03)

Changements dans cette version:

  • support du format de e-book EPUB
  • Ajout d'un menu File/Rename pour renommer le fichier en cours (contribution de Vasily Fomin)
  • support des fichiers TIFF multi-pages
  • support des images TGA
  • support pour plusieurs metadata des fichiers CBZ (bandes dessinées)
  • support pour les images JPEG XR (disponible dans Windows Vista et supérieur, pour Windows XP le Windows Imaging Component doit être installé)
  • the installer is now signed
2.0.1 (2012-04-08)

Changements dans cette version:

  • correction pour le chargement des fichiers .mobi depuis la ligne de commandes
  • Correction d'un bug pour le chargement multiple de fichiers .mobi en une fois
  • Correction d'un bug lors de l'affichage de bulles d'aide pour les entrées de la table des matières
2.0 (2012-04-02)

Changements dans cette version:

  • support pour le format d'e-Book MOBI
  • support pour l'ouverture de documents CHM sur des lecteurs réseau
  • Une sélection sous forme d'image peut être copyée dans le presse-papier avec un clic droit de la souris
  • Utilisation de ucrt pour réduire la taille du programme
1.9 (2011-11-23)

Changements dans cette version:

  • support pour les documents CHM
  • support des entrées tactiles (« touch gestures »), disponible sur Windows 7 ou supérieur. Contribution de Robert Prouse
  • Ouvre les fichiers audio et video liés dans un media player externe
  • Support amélioré des groupes de transparence PDF
1.8 (2011-09-18)

Changements dans cette version:

  • Support amélioré des zones de texte dans les formulaires PDF
  • Plusieurs améliorations et correcions de bugs mineures
  • Chargement accéléré de certains types de fichiers djvu
1.7 (2011-07-18)

Changements dans cette version:

  • Favoris
  • Support amélioré des lagages « right-to-left » (de droite à gauche), tels que l'arabe
  • les numéros de pages « logiques » (c.-à-d., non automatiques, comme par exemple i, ii, iii, etc.) sont affichés et utilisés, si un document les fournit
  • Permet de restreindre les fonctionnalités de SumatraPDF avec plus de granularité. Voir sumatrapdfrestric.init pour plus d'informations
  • -named-dest recherché aussi des châines de caractères dans la table des matières
  • Support amélioré des fichiers EPS (nécessite Ghostscript)
  • Installeur plus fiable et plus efficace
  • Nombreuses améliorations et correcions de bugs mineures
1.6 (2011-05-30)

Changements dans cette version:

  • Ajout du support pour l'affichage des documents DjVu
  • Affichage de la liste des documents fréquemment lus quand aucun document n'est ouvert
  • Ajout du support pour l'affichage des documents Postscript (nécessite la présence d'une version récente de Ghostscript)
  • Ajout du support pour l'affichage de répertoires contenant des images glissés dans la fenêtre SumatraPDF
  • Support des liens cliquables ; Table des matières pour les documents XPS
  • Affiche la progression de l'impression et permet son annulation
  • Ajout d'un bouton d'impression sur la barre d'outils
  • expérimental : prévisualisation de documents PDF dans Windows Vista and 7. Crée des miniatures et affiche les documents dans la prévisualisation du panneau d'exploration. Doit être explicitement sélectionné durant l'installation. Des comptes-rendus indiquent que ça ne fonctionne pas avec Windows 7 x64.
1.5.1 (2011-04-26)

Changements dans cette version:

  • Améliorations de quelques bugs
1.5 (2011-04-23)

Changements dans cette version:

  • Ajout du support pour l'affichage des documents XPS
  • Ajout du support pour l'affichage des bands dessinées CBZ et CBR
  • Ajout d'un menu contextuel pour la copie de texte, d'adresses de liens et de commentaires, et pour les plugins navigateurs, des fonctions d'enregistrement et d'impression.
  • Ajout du parcours du répertoire en cours (Ctrl+Shift+Right ouvre le PDF suivant, Ctrl+Shift+Left ouvre le précédent)
1.4 (2011-03-12)

Changements dans cette version:

  • Plugin pour les navigateurs Internet Firefox/Chrome/Opera (Internet Explorer non supporté). Par défaut, non installé.
  • IFilter, qui améliore la recherche de texte dans les fichiers PDF dans la fonction de recherche intégrée de Windows accessible dans le menu démarrer ou le menu contextuel de l'Explorateur Windows. N'est pas non plus installé par défaut
  • Possibilité de choisir le répertoire d'installation avec l'installeur
  • Options de menu pour réouvrir le document courant dans Foxit et PDF-XChange (s'ils sont installés)
  • Nous ne compressons plus l'exécutable dans l'installeur avec mpress. Il produit une identification erronée de virus dans Sumatra par certains programmes anti-virus. L'inconvénient est que le programme occupe plus d'espace disque, désormais. Note: nous compressons toujours la version portable dans le .zip
  • Suppression du paramètre de ligne de commandes -title
  • Support pour les fichiers PDF encryptés AES-256
  • Correction d'un dépassement d'entier signalé par Jeroen van der Gun et quelques autres corrections mineures et améliorations pour la prise en charge des PDF
1.3 (2011-02-04)

Changements dans cette version:

  • Sélection et copie de texte améliorée. Nous imitons désormais la méthode dont un navigateur Internet ou Adobe Reader travaillent : simplement sélectionner le texte avec la souris et utiliser Ctrl-C pour le copier dans le Presse-papier
  • Shift+Bouton gauche de la souris permet maintenant de faire défiler le document, Ctrl+Bouton gauche crée toujours une sélection rectangulaire (pour copier une image)
  • le raccourci 'c' est une bascule en mode Vue continue/discontinue
  • '+' / '*' sur le pavé numérique permettent le zoom et la rotation
  • Ajout d'icônes dans la barre d'outils pour "Page unique" et "Pleine largeur", et mise à jour de l'apparance des icônes de la barre d'outils
  • support pour la molette de défilement de la souris pour le défilement avant/arrière
  • 1.2 introduit un nouveau mode plein écran qui devient le mode plein écran par défaut. L'ancien mode restait disponible mais peu aisément accessible. Nous avons ajouté une option de menu Affichage/Présentation menu pour le nouveau mode et Affichage/Plein écran pour l'ancien, pour rendre le premier plus visible
  • Installeur nouveau et amélioré
  • Performance du zoom améliorée (zoomer à 6400% ne fait plus planter le programme)
  • La recherche de texte utilise moins de mémoire vive
  • Nouvelles améliorations pour l'impression
  • Mise à jour des traductions
  • Mise à jour avec la plus récente version de mupdf pour diverses améliorations et corrections de bugs
  • Utilisation de la librairie libjpeg-turbo à la place de libjpeg, pour un décodage plus rapide de certains PDFs
  • Mise à jour de la librairie openjpeg avec la version 1.4 et de freetype avec la version 2.4.4
  • Correction de deux problèmes de dépassement de calcul signalés par Stefan Cornelius de Secunia Research

Participants à cette version :

  • Krzysztof Kowalczyk
