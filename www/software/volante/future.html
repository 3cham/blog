Volante : Volante Developer's Guide : Future of Volante

Volante is feature-rich, fast and stable.

My focus with Volante is reliability and documention. Most of my effort is spent writing documentation, automated test framework, more tests and fixing bugs exposed by tests.

Adding new features has lower priority. New features will be added based on feedback from people using Volante in real software. You can use the bug tracker to request new features. When requesting new features please also state why the feature is valuable, not just describe what it should do.

Volante is an open-source project. The source is at github.com/kjk/volante and I welcome particiapation in the project. If you want to contribute the code, the usual rules apply: fork the repo, make the changes, send a pull request.
