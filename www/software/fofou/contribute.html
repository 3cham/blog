How to contribute to Fofou?

Fofou is open-source software (combination of Public Domain and BSD) hosted on GitHub.

Feel free to use and abuse it in any way you want.

If you want your changes folded into fofou sources, point me to your git branch of the sources, explain why the changes are good and if you sound convincing, I'll merge them.

Fofou is good enough to be used - I'm using it for hosting discussions for my open-source projects and it's working out great.

Of course, there's always something to improve. There's a todo list in the source, which lists my wishes. I'm sure you'll have no trouble coming up with your own list of possible improvements.
