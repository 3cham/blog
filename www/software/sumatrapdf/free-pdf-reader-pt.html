Sumatra PDF

Um leitor de PDFs para Windows



O que é o Sumatra PDF?

O Sumatra PDF é um programa para sistemas Windows e funciona como leitor de PDF, XPS, DjVu, CBZ e CBR. É rápido, gratuito e de código livre. Funciona em qualquer computador com Windows.

Qual a necessidade de outro leitor de PDFs?

O Sumatra possui uma interface minimalista. Privilegiamos a simplicidade ao invés de muitas funcionalidades.

É pequeno e com um arranque muito rápido.

Foi desenvolvido para poder ser usado em qualquer lado: apenas 1 ficheiro de modo a que o consiga executar num disco USB. Não produz efeitos no registo do sistema.
