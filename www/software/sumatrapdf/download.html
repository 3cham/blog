Problems downloading?

Downloads only work if your browser properly sends http referer header. All browsers do it by default although you can manually disable it. You can try another browser or re-enable sending referer header.

Re-enabling referer header in Firefox/Mozilla:

  • type about:config in url field
  • find network.http.sendRefererHeader setting and change it to 2

Also some firewalls (like Sunbelt (Kerio) Firewall) might block sending referer header.
