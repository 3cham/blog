Translating Sumatra PDF web pages

You can help make Sumatra PDF more popular in your country by translating the most important HTML pages on this website to your language.

Sumatra's web pages are just HTML files so you'll need to understand basics of HTML.

You'll also need to understand basics of git (git tutorial since the pages are managed in a git repository.

Here's the process of contributing a translation:

  • fork the sources of this website
  • Sumatra's web pages are in www/software/sumatrapdf directory
  • translate a page and save it with a prefix for your language e.g. manual.html translated to French should be named manual-fr.html.
  • commit the translation to your fork
  • send me pull request
  • I'll merge the changes and publish them to a website

It's up to you which pages to translate. The most important pages are:

  • free-pdf-reader.html
  • manual.html (note: keyboard shortucts might be different than in Engilsh version, so please double-check!)
  • download-free-pdf-viewer.html

Please don't translate the following pages:

  • bugs.html
  • develop.html
Those are instructions on how to contribute or submit bugs. Sumatra team is not able to handle bug submission or contributions that are not in English and translating those pages could set an expectation to the contrary.
